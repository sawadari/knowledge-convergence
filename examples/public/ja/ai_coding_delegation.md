# 例：AIコーディング委任

## 状況

チームが、AIコーディングエージェントにリポジトリ内の変更実装を依頼したいと考えています。

依頼内容は機能アイデアに基づいていますが、要求は完全には承認されていません。

## 行動候補

```text
AIコーディングエージェントに、支払い検証モジュールを変更させる。
```

## 必要な確認

- 要求は承認済みか
- 設計判断は記録されているか
- エージェントは対象リポジトリを編集してよいか
- ツール範囲は制限されているか
- データアクセス範囲は定義されているか
- 人間レビューゲートは必要か
- ロールバック経路は定義されているか
- テスト戦略は定義されているか

## 結果

```text
outcome = hold
reason = approved requirement, authority envelope, and rollback path are missing
```

## 教訓

AIコーディングの速度は、上流判断の成熟度と委任統治の必要性を消しません。
