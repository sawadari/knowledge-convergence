# 例：組織のレビュー能力不足

## 状況

高リスクのアーキテクチャ判断には、安全専門家のレビューが必要です。

安全専門家は適切な能力と権限を持っています。しかし、リリースゲート前にレビューする時間を確保できません。

## 判断候補

```text
リリースゲート前にアーキテクチャ案Xを承認する。
```

## 知識収束チェック

| 次元 | 状態 |
|---|---|
| 意味 | 明確 |
| 根拠 | 一部あり |
| 責任 | 安全専門家は特定済み |
| レビュー能力 | 不十分 |
| ドメイン妥当性 | 必要レビュー不足によりブロック |

## 結果

```text
outcome = escalate
reason = review capacity gap for high-risk decision
```

## 教訓

役割が割り当てられていることと、意味あるレビュー能力が利用可能であることは同じではありません。
